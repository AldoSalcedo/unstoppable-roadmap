## 1. JOB APPLICATION FUNNEL
### From 100 Applications → 1-4 Offers

```
100 Applications Sent
       ↓
       ↓ 30% Response Rate (30 companies view)
       ↓
30 Screening Interviews (HR Round)
       ↓
       ↓ 60% Pass Rate (18 advance)
       ↓
18 Phone Screens (30 min, recruiter/manager)
       ↓
       ↓ 50% Pass Rate (9 advance)
       ↓
9 Technical Interviews (live coding, 60-90 min)
       ↓
       ↓ 40% Pass Rate (3-4 advance)
       ↓
3-4 Final Rounds (leadership, culture fit, offer discussion)
       ↓
       ↓ 70% Pass Rate (2-3 offers)
       ↓
2-3 Job Offers
       ↓
1 Accepted Offer ← YOU WIN
```

**Key Conversion Rates**:
- Apply → Phone screen: 30% (volume matters)
- Phone → Technical: 60% (preparation matters)
- Technical → Final: 40% (system design + culture matters)
- Final → Offer: 70% (negotiation matters)

**Implication**: To get 1 offer, you need ~100 applications.

**Volume Strategy**:
- Week 15: 10 applications (high quality, personalized)
- Week 16: 40+ applications (lower personalization, higher volume)
- Total: 50+ applications by end of Week 16

**Quality Strategy**:
- Customize resume to job description (10 min)
- Write 1-2 sentence personalized cover letter (5 min)
- Research hiring manager + company tech stack (5 min)
- Apply + track (2 min)
- = 22 minutes per application (realistic speed)

## 2. SALARY NEGOTIATION FRAMEWORK
### BATNA, Anchoring, Package Optimization

```
┌─────────────────────────────────────────────────┐
│           NEGOTIATION FRAMEWORK                 │
├─────────────────────────────────────────────────┤
│                                                 │
│  Step 1: Define Your BATNA (Best Alternative)  │
│  ─────────────────────────────────────────────   │
│    BATNA = Freelance income ($75/hr × 30 hrs = $3.9k/mo)   │
│    Minimum acceptable: $70k/year US or 100k MXN/month      │
│    Ideal target: $85-100k/year US or 140k+ MXN/month       │
│                                                 │
│  Step 2: Research Market (Anchoring Data)       │
│  ─────────────────────────────────────────────   │
│    Levels.fyi: Zocdoc E4 = $120-140k TC               │
│    Glassdoor: Oscar Health SWE = $95-115k TC          │
│    Blind: Epic Dev = $80-100k (slower growth)         │
│    → Your anchor: $95-110k (upper middle quartile)     │
│                                                 │
│  Step 3: Let Them Anchor First                  │
│  ─────────────────────────────────────────────   │
│    Them: "We can do $75k"                         │
│    ✓ Counter: "I was looking at $90k+" (up 20%) │
│    ✗ Accept first offer (leaves $15k on table)     │
│                                                 │
│  Step 4: Optimize Full Package                  │
│  ─────────────────────────────────────────────   │
│    If base is stuck at $80k:                     │
│    - Can we add signing bonus? (+$5-10k)        │
│    - What's equity vesting? (4-year @ 1% = $30k+)  │
│    - Bonus structure? (15-20% = $12-16k/year)   │
│    - PTO, health, 401k match?                   │
│    → Total package: $80k + $10k sign + $15k bonus │
│                     + equity = $105k+ equivalent    │
│                                                 │
│  Step 5: Create Urgency (Compress Timeline)     │
│  ─────────────────────────────────────────────   │
│    "I have another offer due by Friday"           │
│    (Use if true; shows urgency)                 │
│    "I want to commit immediately once we        │
│     align on terms"                             │
│    → They accelerate decision                   │
│                                                 │
└─────────────────────────────────────────────────┘
```

**Key Principles**:
1. ✅ **Never accept first number** — Counter-offer always
2. ✅ **Get market data** — Know ranges before negotiating
3. ✅ **Think total comp** — Salary + bonus + equity + benefits
4. ✅ **Anchor high** — They'll counter, meet in middle
5. ✅ **Be creative** — If salary fixed, negotiate other terms
6. ✅ **Use leverage** — "I have another offer..." (if true)
7. ✅ **Move fast** — Compress timeline once close

**US Market Ranges** (Mid-level SWE, healthcare tech):
- Base: $75-95k
- Bonus: 10-20% (7.5-19k)
- Equity: 0.25-1.5% (varies, startup equity can be worthless)
- Benefits: Health, 401k match, PTO

**Total Comp Target**: $95-115k USD/year

## 4. TECHNICAL INTERVIEW PATTERNS
### Behavioral + Live Coding + System Design

```
┌──────────────────────────────────────────────────┐
│      INTERVIEW ROUNDS (Typical Flow)             │
├──────────────────────────────────────────────────┤
│                                                  │
│  Round 1: SCREENING (30 min, Recruiter)        │
│  ────────────────────────────────────────────    │
│  - Tell me about yourself (30 sec)              │
│  - Why this company? (1 min)                    │
│  - Brief work history + relevant projects       │
│  - Logistics: When can you start? Visa needs?   │
│  → Goal: Confirm basic fit, technical skills exist
│                                                  │
│  Round 2: TECHNICAL INTERVIEW (60 min, Engineer)│
│  ────────────────────────────────────────────    │
│  - Live coding problem (45 min)                 │
│    * Easy-Medium LeetCode problem               │
│    * Talk through approach                      │
│    * Write pseudocode first                     │
│    * Code on whiteboard or IDE                  │
│    * Test with examples                         │
│  - Behavioral Q (15 min)                        │
│    * "Tell me about a hard problem you solved" │
│  → Goal: Can you code? Communication?           │
│                                                  │
│  Round 3: SYSTEM DESIGN (60 min, Senior Eng)   │
│  ────────────────────────────────────────────    │
│  - Design [system] (e.g., appointment booking)  │
│  - Functional requirements (what should it do?) │
│  - Non-functional requirements (scale, latency) │
│  - High-level architecture (components)         │
│  - Deep dive (pick 1 component, go deep)       │
│  - Tradeoffs (what would you change?)           │
│  → Goal: Can you architect? Think at scale?     │
│                                                  │
│  Round 4: FINAL ROUND (60 min, Manager/Leader) │
│  ────────────────────────────────────────────    │
│  - Deep behavioral Q (30 min)                   │
│    * Leadership, conflict resolution, growth    │
│  - Product thinking (15 min)                    │
│    * "How would you improve [company product]?"│
│  - Offer discussion (15 min)                    │
│    * What are you looking for?                  │
│    * Compensation expectations?                 │
│  → Goal: Culture fit? Leadership potential?     │
│                                                  │
└──────────────────────────────────────────────────┘
```

**Interview Success Checklist**:

```
BEFORE INTERVIEW:
☐ Research company (mission, products, recent news)
☐ Research interviewer (LinkedIn, GitHub, blog posts)
☐ Review job description 3 times
☐ Prep 5 core stories (see "BEHAVIORAL STORIES")
☐ Practice 3-5 LeetCode medium problems
☐ Get whiteboard or pen + paper ready
☐ Test video/audio 15 min early

DURING TECHNICAL:
☐ Repeat problem back to interviewer (confirm you understood)
☐ Ask clarifying questions (scale? constraints? edge cases?)
☐ Think out loud (show your process, not just answer)
☐ Write pseudocode first, then code
☐ Test your code with examples
☐ Discuss tradeoffs (time vs space, simple vs complex)
☐ Ask questions at end (shows engagement)

AFTER INTERVIEW:
☐ Send thank you email within 24 hours
☐ Mention specific thing you discussed (shows attention)
☐ Reiterate interest in role/company
☐ Ask about next steps, timeline
```

**Core Behavioral Stories** (Prepare 2-3 min versions):

1. **Tell me about yourself**
   - Background (biology + software)
   - Key projects (portfolio)
   - Why healthcare tech
   - Why now (job search)

2. **Hardest technical problem**
   - [Real problem from portfolio]
   - What you learned
   - How you apply that learning now

3. **Failure + learning**
   - [Real mistake from learning journey]
   - What you learned
   - How you changed approach

4. **Why [company]?**
   - Specific product feature/problem that excites you
   - How your background fits their mission
   - Concrete reason (not "it's a great company")

5. **Bilingual advantage**
   - Why Spanish + English is valuable for healthcare expansion
   - Culture/communication benefits
   - LATAM market opportunity
