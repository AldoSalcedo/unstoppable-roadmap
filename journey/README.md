# Mi Viaje: 16 Semanas hacia $100k+ MXN/mes

## ¿Qué es esto?

Esta carpeta es tu **Obsidian vault personal** para rastrear los 16 semanas de tu transformación como ingeniero de software en healthcare. Combina:
- **Sprints semanales** (TypeScript, DS&A, certs)
- **Logs diarios** para mantener la cadena viva
- **Certificaciones** (AWS Developer, Azure AI Engineer)
- **Reflexiones** sobre cómo tu background (Auditoría/QBP/Ventas) te diferencia

## Cómo usar esta vault

1. **Abre en Obsidian**: `File → Open Vault → /path/to/journey/`
2. **Cada mañana**: Ejecuta `/daily-standup` para generar tu entrada del día
3. **Al final del día**: Completa `journey/DAILY-LOG.md` con tus wins y blockers
4. **Fin de semana**: Llena `journey/weeks/week-XX/REVIEW.md`
5. **Links rápidos**: Los archivos markdown están entrecruzados para fácil navegación
